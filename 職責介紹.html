職責介紹

主攻 副攻 舉球員 自由球員
主攻手
    職責介紹
  1. 球場上的主要攻擊手
  2. 防守反擊中的主要得分球員
  3. 擔任一傳的重任
  4. 一般站在4號位，或在跑位時換到4號位
    球員要求
  1. 球員身材高大
  2. 跳躍力強，擁有強勁的扣殺力
  3. 擅長強攻、突破對方防禦
  4. 擅於扣調整球、戰術球
    代表球員
副攻手
    職責介紹
  1. 以快攻為主要攻擊方式
  2. 以快攻掩護隊友組成各種快速多變的進攻方式
  3. 防守攔網以阻止對方的快攻戰術
  4. 一般站在3號位，或在跑位時換到3號位
    球員要求
  1. 球員身材高大
  2. 衝擊跳躍力強、變相跑動能力強
  3. 擅於攔網、配合兩側隊友攔網
    代表球員
舉球員
    職責介紹
  1. 又稱二傳手、拖手
  2. 通常一個球隊配備1~2名舉球員
  3. 球隊中擔任二次傳球組織進攻的球員
  4. 場上組織進攻、實施戰術的組織者
  5. 發揮全體隊員的特點以及組織隊伍的進攻力量
  6. 一般站在2號位，或在跑位時換到2號位
    球員要求
  1. 良好的心理素質
  2. 細膩的手上技術
  3. 快速的腳下移動
  4. 攔網能力強
  5. 擅於隨機應變、團隊合作
    代表球員
自由球員
    職責介紹
  1. 防守反擊中的後排組織防守人員
  2. 無須向裁判請求許可，可隨時取代後排任一隊員出賽
  3. 擔任接起一傳和防守的重要任務
  4. 需做出精準的卡位判斷和良好的手感，為防守反擊創造機會
  5. 不可參與進攻
  6. 著裝顏色需異於其他球員，以便確認
    球員要求
  1. 矮小
  2. 靈活
  3. 移動速度快
  4. 應變能力強
  5. 穩定的一傳到位率
    代表球員
